
ReadMe file for sql cabinet upgradation from OmniFlow 6.0.2 to OmniFlow 6.1.2

Prerequisites:-
Important : SQL Server client is required on the m/c, where from this batch file is to be executed.

Procedure to be followed:-

1. Open the file upgrade.bat in notepad.
2. Specify the values of the following parameters:-

	UPGRADE_SERVER		for MsSql Server Name  
	UPGRADE_CABINET		for Database name  
	UPGRADE_USER		for MsSql Server User name  
	UPGRADE_PWD		for MsSql Server Password - can be blank

3.Save and close the file.
4.Double click on upgrade.bat to upgrade the cabinet.
5.When the message "Press any key to countinue... ",comes,press any key to continue.
6.Open output.txt for viewing the output of cabinet upgradation.
